Shadow clone jutsu

Shadow Clone Jutsu (Kage Bunshin no Jutsu) is a forbidden technique that lets the user create copies of themselves. The clones created by the Shadow Clone Jutsu are so realistic that it not only creates an illusion of a clone, but an actual solid clone that duplicates everything, including their clothing.

Rasengan

The Rasengan in one of the most iconic Jutsu in Naruto that's been passed down from generations in Konohagakure. This Jutsu allows the user to manifest a ball of chakra that spins at extremely high speed and generates enough power to rip through whatever it touches

Summoning Jutsu

Summoning Jutsu is a contract that is placed between the caster of the Jutsu and a target. The target is usually an animal but there are a few exceptions. These summons act like Familiars and will fight alongside you in battle.
